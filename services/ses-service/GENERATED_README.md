# AWS SES FlowRunner Extension

| **Kind** | **Name** | **Description** |
|----------------|----------------|---------------------|
| ACTION | Send Email | Sends a plain text or HTML email to one or more recipients via a verified sender address. Supports optional CC, BCC, and reply-to addresses. |
| ACTION | Send Templated Email | Sends an email using a pre-created email template, with optional template data for variable substitution. |
| ACTION | Send Bulk Templated Email | Sends a templated email to multiple recipients in a single batch operation, with optional per-recipient variable substitution. |
| ACTION | Create Email Template | Creates a reusable email template with a name, subject, and optional text and HTML body parts. Templates support Handlebars-style variable substitution. |
